# 🎮 Tic Tac Toe – React Edition

A modern, sound-popping, emoji-flashing twist on the classic Tic Tac Toe game — built with React, styled with pure CSS, and fueled by fun.

---

## ✨ Features

- ❌ **X vs O** – Custom avatars for each player (`❌` and `⭕`)
- 🔊 **Click & Win Sounds** – Audio feedback for every move and win
- ⚡ **Animated Gameplay** – Smooth transitions and subtle effects
- 🎯 **Win Detection** – No cheating allowed!
- 🔄 **Reset Button** – Play again, and again… and again

---

## 🚀 Getting Started

### 1. Clone the repo
```bash
git clone git@github.com:RamseyAndrew/TIC-TAC-TOE.git
cd tic-tac-toe-react
````

### 2. Install dependencies

```bash
npm install
```

### 3. Start the game

```bash
npm run dev
```

Visit `http://localhost:5173` and start tapping!
